This page contains draft store copy for the official Re:T-UI Play Store release.

## Short Description

Terminal-style Android launcher with deep theming, presets, modules, and command-first control.

## Full Description

Re:T-UI is a terminal-style Android launcher for people who want their home screen to feel alive, personal, and a little bit hacker-adjacent without giving up daily usability.

It keeps the command line at the center of the experience while adding modern quality-of-life features that make it practical on current Android devices.

With Re:T-UI you can:

- launch apps and actions from a terminal prompt
- build your own command vocabulary with aliases
- organize apps into groups and browse them in a terminal-style drawer
- style the launcher with custom fonts, presets, and wallpaper-derived colors
- view notifications in a themed on-screen terminal panel
- use a music visualizer that fits the desktop instead of fighting it
- customize behavior through both a settings hub and editable config files

Re:T-UI is made for users who want:

- a command-first launcher
- strong visual identity
- deep customization
- fast interaction once muscle memory kicks in

Highlights:

- terminal-style launcher UI
- app drawer with groups and alphabet navigation
- preset save/apply workflow
- wallpaper auto-color
- notification terminal
- music visualizer
- alias support
- editable configuration files for power users
- terminal-style settings hub

This Play Store version is the official stable release and the easiest way to support ongoing development.

GitHub and Firebase may offer source access and testing builds, but the Play Store build is the main polished channel for everyday users.

If you want a launcher with personality, not just another grid of icons, Re:T-UI is for you.
