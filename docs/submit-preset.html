Share a Re:T-UI look for marketplace review.

Uploads go into a moderation queue first. The gallery uses real screenshots so people can judge the look before downloading anything.

Create The File

  1. Customize Re:T-UI until the look is ready.
  2. Open Settings, then System & Support.
  3. Choose Create Shareable Configuration.
  4. Pick Current Active Look or one of your saved presets.
  5. Save the exported file and upload it here with a screenshot.

What Gets Published

  • The screenshot you upload.
  • The visual shareable configuration package for download.
  • The app version if it exists in the preset manifest.
  • The wallpaper image only if you attach one separately.
